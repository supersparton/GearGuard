# GearGuard

Equipment management and request tracking system built with React and Supabase.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables in `.env`:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

3. Run development server:
```bash
npm run dev
```

## Structure

- `frontend/src/core/equipment/` - Person 1
- `frontend/src/core/kanban/` - Person 2  
- `frontend/src/core/dashboard/` - Person 3
- `frontend/src/shared/` - Shared components
